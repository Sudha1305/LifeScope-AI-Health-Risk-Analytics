# 🚀 LifeScope AI: Lifestyle-Based Health Risk Analytics & Scoring System
## 📌 Overview This project analyzes lifestyle factors such as sleep duration, stress levels, physical activity, and daily habits to understand their impact on health risk.
## 🎯 Objectives - Analyze behavioral health patterns - Identify relationships between lifestyle and health indicators - Develop a risk scoring system
## 🛠 Tools & Technologies - Python - Pandas - Seaborn - Google Colab 
## 📊 Project Workflow 
   1. Data Cleaning 
   2. Exploratory Data Analysis (EDA)
   3. Data Visualization
   4. Risk Score Model Development. 
## ⚡ Key Features - Custom health risk scoring system - Classification into Low, Moderate, High risk - Visual insights using graphs. 
## 💡 Key Insights - Lower sleep duration increases stress levels - Higher physical activity improves overall health - Lifestyle patterns significantly influence health risk.
## 🚀 Conclusion This project demonstrates how data analytics can be used to extract meaningful insights from lifestyle data and support health awareness
